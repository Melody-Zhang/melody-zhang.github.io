Posted on 2021-04-18 Edited on 2021-04-19

过程

pointer -> uintptr -> 运算 -> pointer -> 解引用
&sth -> unsafe.Pointer() -> uintptr() -> unsafe.Pointer() -> *sthptr -> *

go中存储的内存大小

int8 - 1byte

int - 8bytes

string - 16bytes

noted

只有 uintptr 类型可以执行加减操作
+1 意味着加一个 byte
那么该数据结构占用多少byte就要相应加多少byte
如：
int8 +1
int32 +4
int64 +8
string +16
